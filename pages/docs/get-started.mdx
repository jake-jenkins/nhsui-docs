# Get Started

Adding NHS UI Components to an app or site is simple by adding some lines into your template file.
NHS UI Components can be used indepentanly or on top of any other framework, such as Bootstrap etc.

---

### Add to your site

1. Add stylesheet to your header.

2. Add javascript to body.

3. Start adding classes to your code to use components.

```html
// Add inside <head> tag
<link rel="stylesheet" href="https://nhs.pages.dev/nhs.css">
```

```html
// Add before </body> tag
<script src="https://nhs.pages.dev/nhs.js"></script>
```
